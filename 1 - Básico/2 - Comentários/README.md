#Comentários
Como dito no arquivo, embora seja algo simples, para o iniciante, não é algo tão óbvio ou importante, mas é uma das formas e talvez, a mais
utilizada para documentar um código, como um manual, indicando para que serve cada etapa, mesmo que já haja um padrão com nomes de variáveis,
funções e etc, é importante para não criar confução e auxiliar em uma compreensão rápida e tranquila.

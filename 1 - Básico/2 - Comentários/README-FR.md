# Commentaire
C'est d'une importance, faire des commentaires dans un code, parce que dans ce code c'est nécessaire de comprendre tout ce qui est écrit, 
ou créé dans ce code, peu importe qu'il y ait un patron dans la variable, dans la fonction,
mais pour que d'autres personnes et toi aussi comprennent ce code, c'est nécessaire de faire des commentaires.

# Console.log()
É uma ferramenta no JavaScript para depurar o código, que quer dizer visualizar o valor das variáveis e os resultados de expressões,
ajuda a rastrear a execução do código e quais partes do script é executável.
Além de exibir informações úteis, permite inspecionar a estrutura de objetos e arrays complexos.

C'est un utilitaire dans le JavaScript, pour faire le débugage, qui sert à visualiser la valeur des variables et les résultats
des expressions, à aider pour savoir l'exécution dans le code et ce qui part dans le code, cette exécution.
Non seulement montrer les informations utiles, mais permettre l'inspection dans 
la structure de l'objet et de l'array avancé. Il y a d'autres consoles en Javascript que nous ne voyons plus aller.
